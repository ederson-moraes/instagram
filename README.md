## 🚀 Technologies

- React
- NodeJS
- HTML
- JavaScript
- CSS

- React Router
- Zustand

## 📦 Features

- Photo feed with posts from all users
- Like on posts
- User profile page
- Upload photos and captions
- create hashtags
- descripe Place
- Responsive UI

## 🛠️ Getting Started

### Prerequisites

- Node.js (>=18)
- React
- React Native

### ⚙️ Local Installation

```bash
git clone https://github.com/ederson-moraes/instagram.git
cd backend
npm install
npm run dev
```
